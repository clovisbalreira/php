# Repositório de projetos em PHP

## 📌 Descrição
Repositório com exercícios e projetos feitos em PHP.

## 🧠 Conteúdo
- Estruturas de controle (if, switch, loops)
- CRUD e operações com banco de dados
- Projetos simples e exemplos práticos
- Aplicações web com formulários e integração

## ▶️ Como rodar
1. Coloque a pasta do projeto no diretório do servidor local (por exemplo, `htdocs` do XAMPP).
2. Inicie o servidor Apache e MySQL.
3. Acesse pelo navegador em `http://localhost/` seguido da pasta do projeto.
4. Abra os arquivos `.php` desejados no navegador.

## 📝 Observação
Alguns projetos podem exigir configuração de banco de dados ou dependências específicas. Verifique os arquivos de cada pasta para instruções detalhadas.
