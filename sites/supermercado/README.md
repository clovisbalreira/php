# Supermercado

## 📌 Descrição
Projeto de sistema de supermercado em PHP com controle de produtos e vendas.

## 🚀 Tecnologias
- PHP
- HTML
- CSS

## 🎯 Funcionalidades
- Listagem de produtos
- Controle de estoque
- Interface de vendas simulada

## ▶️ Como rodar
1. Clone o repositório
2. Coloque a pasta do projeto no diretório do servidor local (por exemplo, `htdocs` do XAMPP)
3. Abra o arquivo `index.php` no navegador
