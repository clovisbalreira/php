# Loja

## 📌 Descrição
Projeto de loja virtual em PHP com produtos, carrinho e layout de e-commerce.

## 🚀 Tecnologias
- PHP
- HTML
- CSS

## 🎯 Funcionalidades
- Exibição de produtos
- Adição ao carrinho
- Navegação por categoria

## ▶️ Como rodar
1. Clone o repositório
2. Coloque a pasta do projeto no diretório do servidor local (por exemplo, `htdocs` do XAMPP)
3. Abra o arquivo `index.php` no navegador
