# Carrinho

## 📌 Descrição
Projeto de carrinho de compras em PHP com gerenciamento de clientes e pedidos.

## 🚀 Tecnologias
- PHP
- HTML
- CSS

## 🎯 Funcionalidades
- Listagem de itens no carrinho
- Cadastro e edição de clientes
- Gerenciamento de pedidos

## ▶️ Como rodar
1. Clone o repositório
2. Coloque a pasta do projeto no diretório do servidor local (por exemplo, `htdocs` do XAMPP)
3. Abra o arquivo `index.php` no navegador
