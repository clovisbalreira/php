# CRUD

## 📌 Descrição
Projeto demonstrando operações CRUD em PHP para criação, leitura, atualização e exclusão de registros.

## 🚀 Tecnologias
- PHP
- HTML
- CSS

## 🎯 Funcionalidades
- Criação de registros
- Edição de informações existentes
- Exclusão de dados
