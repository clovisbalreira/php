# Estudonauta

## 📌 Descrição
Projeto educativo com exercícios, login de usuário e páginas de conteúdo para estudantes.

## 🚀 Tecnologias
- PHP
- HTML
- CSS

## 🎯 Funcionalidades
- Sistema de login e cadastro
- Páginas de conteúdo educacional
- Interface organizada e responsiva
