## 📌 Descrição
Projeto com exercícios básicos de PHP e lógica de programação voltados para iniciantes.

- HTML
- CSS

- Exercícios de lógica básica

## ▶️ Como rodar
1. Clone o repositório

3. Abra o arquivo `index.php` no navegador
