# MySQL

## 📌 Descrição
Projeto com exemplos de consultas e operações MySQL integradas em PHP.

## 🚀 Tecnologias
- PHP
- HTML
- MySQL

## 🎯 Funcionalidades
- Consultas SELECT, INSERT, UPDATE e DELETE
- Exemplos de joins e agregações
- Resultados exibidos em página web

## ▶️ Como rodar
1. Clone o repositório
2. Configure o banco de dados MySQL
3. Coloque a pasta do projeto no diretório do servidor local (por exemplo, `htdocs` do XAMPP)
4. Abra o arquivo `index.php` no navegador

## 📚 Aprendizados
Aprendi a conectar PHP ao MySQL e a executar consultas SQL em páginas dinâmicas.
