# Temperatura

## 📌 Descrição
Projeto com conversor de temperatura em PHP e exemplos de cálculos simples.

## 🚀 Tecnologias
- PHP
- HTML
- CSS

## 🎯 Funcionalidades
- Conversão de Celsius para Fahrenheit
- Conversão de Fahrenheit para Celsius
- Interface simples para entrada de valores
