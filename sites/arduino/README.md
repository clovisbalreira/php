# Arduino

## 📌 Descrição
Projeto com exemplos e exercícios relacionados ao Arduino integrados com PHP/HTML.

## 🚀 Tecnologias
- PHP
- HTML
- CSS

## 🎯 Funcionalidades
- Exemplo de integração com Arduino
- Apresentação de dados em página web
- Organização de conteúdo didático

## ▶️ Como rodar
1. Clone o repositório
2. Coloque a pasta do projeto no diretório do servidor local (por exemplo, `htdocs` do XAMPP)
3. Abra o arquivo `index.php` no navegador

Aprendi a estruturar projetos web em PHP e trabalhar com exemplos de Arduino.
